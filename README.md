# 🎟️ Lottery DApp

## Features

- **Implemented Paying With Crypto**: You can pay with ethereum to participate in a lottery cycle.
- **Random Winner**: Winner is picked randomly and instantly gets eth as reward.

## Built With

- `Solidity`
- `Hardhat`
